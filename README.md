# Dynamic Calculator

A simple and responsive calculator web application developed using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations through an interactive user interface and demonstrates JavaScript DOM manipulation, event handling, and dynamic expression evaluation.

## Features

- Basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Decimal number support
- AC (Clear All) button
- DEL (Delete Last Character) button
- Dynamic input display
- Responsive and user-friendly interface
- Real-time calculation using JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## Concepts Covered

- DOM Manipulation
- Event Listeners
- JavaScript Functions
- String Manipulation
- Expression Evaluation
- Responsive UI Design
